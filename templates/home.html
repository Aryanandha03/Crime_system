About the Project

Crime Reporting & Case Tracking System is a secure web application developed using Django that enables digital crime reporting, case management, and transparent communication between citizens, police officers, and administrators.

Who Can Use This Platform?

Citizen

Register, submit complaints, upload evidence, track complaint status, and download FIR reports.

Police Officer

View assigned complaints, update investigations, add remarks, and close completed cases.

Administrator

Manage complaints, assign police officers, monitor analytics, and generate reports.

Key Features

Complaint Submission

Submit crime complaints online.

Evidence Upload

Upload images and documents.

Track investigation progress anytime.

FIR Download

Download complaint report in PDF.
